# 手势识别小游戏
## 1. 产品介绍
- ```gestureapp```：手势识别程序
- ```run_main.bat```：执行模块
- INSTALL: ```pip install yolov4_face_install/ yolov4_gesture_install/ gestureapp_install/ --no-index```
- UNINSTALL: ```pip uninstall -y yoloface yologesture gesture_recognition```
## 2. 产品说明
- 登录页面需要人脸识别登录
- 对您的手势进行评分，训练您的反应力，看看您能获得怎样的分数吧！
